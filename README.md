# 💇‍♀️ Beauty Salon Management System (C++ + DSA Project)

This project is a console-based Salon Management System developed in C++ using Data Structures and Algorithms (DSA) concepts. It simulates a real-world beauty salon booking platform where users can register, explore services, book appointments, and manage their schedules efficiently.

##  User Features:
### 🔐 User Registration & Login system
### 💅 Browse and select salon services
### 📅 Book appointments with time slot selection
### 🏢 Choose salon branches for booking
### 💰 Apply discounts based on services or offers
### 💳 Payment processing (simulation)
### 🧾 Automatic bill generation
### ❌ Cancel bookings anytime
### ⏳ View waiting list for fully booked slots
##  DSA Concepts Used:
### 📌 Arrays / Vectors → store services, users, bookings
### 📌 Linked List → manage waiting list for slots
### 📌 Queue → handle booking requests in order
### 📌 Stack → manage booking/cancellation history (optional)
### 📌 Searching Algorithms → find users, services, and bookings
### 📌 Sorting Algorithms → organize slots, bookings, or price lists
## ⚙️ Project Highlights:
##### Built using C++ programming language
##### Strong focus on DSA implementation in real-world system
##### Object-Oriented Programming (OOP) concepts used
##### Menu-driven interface for smooth navigation
##### Efficient booking and cancellation system design
##### Real-world salon workflow simulation
## 🎯 Purpose:

This project demonstrates how DSA + C++ concepts can be applied to build a practical management system, improving understanding of data structures, algorithms, and system design.
